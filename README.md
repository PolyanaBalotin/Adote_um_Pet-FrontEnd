Um website com a ideia de ajudar ONGs financeiramente, adotando um pet virtualmente.

## Tecnologias Usadas

* React Hooks;
* NextJS;
* TypeScript;
* Styled Components(Material UI)

## Comandos

Apos o clone utilize npm install para instalar as dependencia e npm start para iniciar.
